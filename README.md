# 📘 Aula de Lógica de Programação

Este repositório contém materiais, exemplos e exercícios desenvolvidos durante as aulas de **Lógica de Programação**.

## 🧠 Objetivo

O objetivo deste repositório é ajudar no desenvolvimento do raciocínio lógico e na compreensão dos fundamentos da programação, servindo como base para qualquer linguagem.

## 📚 Conteúdo

- Conceitos básicos de lógica
- Algoritmos
- Fluxogramas
- Pseudocódigo
- Estruturas condicionais (if/else)
- Estruturas de repetição (for, while)
- Variáveis e tipos de dados
- Exercícios práticos

## 🛠️ Tecnologias Utilizadas

- Pseudocódigo
- JavaScript
